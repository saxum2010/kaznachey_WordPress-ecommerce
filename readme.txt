Перед установкой модуля необходимо зарегистрироваться на сайте kaznachey.ua (раздел "Подключиться"). 
После подписания договора быдут выданы: 
-идентификатор мерчанта; 
-секретный ключ мерчанта; 
Переданные данные будут использованы при настройки модуля

Порядок установки модуля:
1. Выберите меню в консоли WordPress: Плагины / Добавить новый / Загрузить;
2. Выберите файл с архивом (*.zip) и нажмите "Загрузить";
3. В консоли WordPress выберите пункт меню  Плагины / Установленные, нажмите "Активировать";
4. Перейдите в меню Параметры / Магазин / Платежи;
5. Поставьте галочку у названия "kaznachey" и нажмите "Обновление" - произойдет активация мерчанта в WP-eCommerse;
6. Нажмите "Edit" и укажите все необходимые поля переданные при заключении договора:
	- Идентификатор мерчанта - используется для подписи сообщений при переадресации (необходимо получить у менеджера после подписания договора);	
	- Секретный ключ мерчанта - используется для подписи скрытых сообщений на Result URL (необходимо получить у менеджера после подписания договора);	
7. Сохраните настройки нажатием кнопки "Обновление".
